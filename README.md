## The Problem

Setting up a new project shouldn't take days. You shouldn't have to wire up authentication, database connections, and deployment configurations from scratch every time.

**CodeScaffold exists to solve this.**

## The Solution

With CodeScaffold, you describe what you want, and we build it for you. No more boilerplate. No more configuration headaches. Just your ideas, turned into production-ready code.

## Features

### Build What You Want

Choose exactly the components you need:

- **5 Frameworks**: Next.js, Express, FastAPI, Go Gin
- **5 Databases**: PostgreSQL, SQLite, with Prisma or Drizzle
- **4 Auth Methods**: NextAuth, JWT, Clerk, or none
- **5 UI Options**: Tailwind + shadcn/ui, MUI, Ant Design, or none

### Build It Right

We make sure your components work together:

- **Smart Dependencies**: Automatic compatibility checking
- **Real-time Preview**: See your project structure before building
- **Quality Gate**: Automatic code quality checks

### Build It Your Way

- **AI-Powered**: Let AI generate custom code for your needs
- **Zero Config**: Get started in seconds with `--defaults`
- **Offline Mode**: Works without internet connection

## Quick Start

```bash
# Install
npm install -g @qaz154/codescaffold

# Zero config mode
codescaffold compose --defaults

# Interactive mode
codescaffold compose
```

## Philosophy

### Simplicity First

We believe in doing the right thing, not the easy thing. Every feature we add must earn its place.

### Components Over Templates

Instead of rigid templates, we give you building blocks. Mix and match to create exactly what you need.

### Quality is Not Optional

Every project generated by CodeScaffold is production-ready from day one. No compromises.

### AI as Enhancement

AI helps you work faster, but it's never required. CodeScaffold works beautifully without it.
